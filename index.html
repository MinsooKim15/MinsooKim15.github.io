Visual Structure:

Each scene is consist of two parts, "description section" which explains about the data and the "graph section" which shows the detailed graph.
This visual structure is contestant across this narrative visualization project.
Viewers can understand the data by reading the texts in X and Y axis, and description section on top
There are also tooltips and annotations for users to follow, so they can stay focused on the
presented data on each scene.

Scenes:

Scenes contain 1 description section(contains title of the data, description and key message) and 1 graph section(which shows different graph suitable for each key message)
All 3 scenes use a consistent format avoid viewers to get confused.
The annotation template contains texts and pointers to pinpoint the location that I would like to emphasize

Parameters:

My parameters are variables, which is chart years, country name and birth rate of charts and used as options to drill down the chart.

Triggers:

My triggers are button, drop down menu and mouse over. Buttons are used as a connector to the next chart or page to change the parameters.
Dropdown menus are used as a selector for specific country name
Mouseover are the trigger to show the tooltips which show the detailed information above the bubble chart and scatter plot.

Data Source

WDI(World Development Index) data from World Bank was used.
Used "GDP Per capita","GDP","Life expectancy" and "Birth Rate" indicator from the data was used.
Some cleaning of data was preceded to fill Null values and to only show the countries(not some group of country as "OECD Members") on some of the scenes.
